Mind the Gap

This tool splits an input FASTA file into two files, based on the percentage of gaps present at one or more columns in the alignment. If the number of gaps in a column exceeds the threshold (percentage) specified below, all sequences with the gap will be placed into one file (the gaps will be removed first) , and all sequences containing the residue ("insertion") will be placed into a second file. This is repeated for all remaining sequences, with sequences being added to the first to files, so that only two output files are produced.
