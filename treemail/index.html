Pipeline: TreeMail


This pipeline runs selected applications from the PHYLIP package and emails the results to the address given below. Press the "Details" button below for more information about each mode. A current limitation of this implementation is that it will timeout and terminate an analysis after 1 hour. This will probably only be applicable to bootstrap analyses of several hundred long sequences.
